- **环境**：Visual Studio Code 1.8x+ / Windows 或 macOS

## 适用场景

把 JSON 里一批同名键改名、给多行末尾加分号、把若干 `var` 改成 `const`，若逐行改既慢又易漏。多光标让你在同一编辑会话里对多个位置同时输入。

## 核心思路

按住 **Alt**（Windows/Linux）或 **Option**（macOS）再点击，可在任意列放下多个光标；或 **Ctrl+Shift+L** 选中当前单词的所有出现并各自落一个光标；**Ctrl+D** 逐个选中下一个相同词。

## 最小示例

1. 选中单词 `foo`。  
2. 按 `Ctrl+Shift+L`，所有 `foo` 同时被选中。  
3. 直接输入 `bar`，全部替换。  

对列对齐的重复结构：在首行目标列按 `Alt+Click` 竖着点几行，即可纵向矩形编辑。

## 常见坑

- 多光标时误触会点到不想改的行，先 `Esc` 清成单光标。  
- 与 Vim 插件并存时，快捷键可能被占用，需在键盘快捷方式里查冲突。  
- 全文件替换超大词时，建议仍用「在文件中替换」预览 diff，避免误伤字符串字面量。
