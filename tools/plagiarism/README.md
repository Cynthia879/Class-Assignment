# 查重与学术规范（班级协作说明）

本目录提供**流程说明**与**本地辅助脚本**，不替代学校指定的官方查重系统。

## 学校/课程指定渠道（优先）

- 按教务处或课程组要求使用 **知网、维普、万方** 等学校采购的系统。
- 提交前保留**查重报告 PDF** 与论文终稿版本号，便于小组自查及按课程要求存档。

## 写作阶段自检（降低重复风险）

1. **复述与引用**：观点来自他人处必须改写并标注引用；直接引用需加引号并文献标注。
2. **图表与代码**：使用他人图、表、代码须获得授权并注明来源。
3. **自我抄袭**：已发表或已提交作业的内容再提交需按课程规则说明。

## 本地辅助工具

| 工具 | 说明 |
|------|------|
| [`similarity_hint.py`](similarity_hint.py) | 对两个 UTF-8 文本做**非常粗略**的字符级相似度提示，仅作自查参考，**不能**作为查重结论。 |
| `texcount` | TeX Live 自带：`texcount -sum main.tex` 可统计字数。 |

## 使用 similarity_hint.py（可选）

```bash
python similarity_hint.py 论文A.txt 论文B.txt
```

输出为 0--100 的简易相似度估计，**不等同于**任何官方查重率。
